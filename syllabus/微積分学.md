# 授業概要

本講義は、高校数学の知識をベースに、大学レベルの**厳密な数学（微分積分学・解析学）**を体系的に学ぶ内容となっています。

### 1. 解析学の基礎固め
* 直感的な理解に留まっていた「極限」や「連続性」を、**論理記号やε-N論法**を用いて再定義します。
* 集合、写像、距離空間といった現代数学の共通言語を習得し、より抽象度の高い議論（位相やコンパクト性）への橋渡しを行います。

### 2. 微分積分学の深化
* **1変数から多変数へ**：偏微分や重積分へと拡張し、勾配ベクトルやヤコビ行列を用いた解析手法を学びます。
* **理論的証明**：テイラーの定理、陰関数定理、微積分の基本定理など、主要な定理の背景にある数学的構造を精査します。

### 3. 応用と発展
* **極値問題**：ヘッセ行列（2次形式）を用いた判定や、ラグランジュの未定乗数法による条件付き最適化を扱います。
* **物理・工学への繋がり**：オイラーの公式、広義積分（特殊関数）、および線形常微分方程式への線形代数の応用までをカバーします。

### 4. 学習の定着
* 全4回の試験（小テスト・中間・教場試験）とその解説を通じて、計算技術だけでなく、定義に基づいた論理展開の習得度を段階的に確認します。

## 授業計画

| 回 | テーマ | 内容のポイント |
| :--- | :--- | :--- |
| 1-3 | **数学の基礎・論理** | ガイダンス、集合（外延・内包的記述）、論理記号（∀, ∃） |
| 4-6 | **実数と数列の極限** | 実数の連続性、ε-N論法、数列の収束と性質 |
| 7-10 | **写像と関数の極限** | 写像の基礎（単射・全射・逆関数）、関数の極限、連続関数の性質 |
| 11-14 | **距離空間と位相** | 多変数への拡張、開集合・閉集合、連結性、点列コンパクト性 |
| 15-16 | **中間評価①** | 前期前半の小テストと解説 |
| 17-20 | **微分法の基礎と展開** | 微分の諸性質、平均値の定理、テイラーの定理、オイラーの公式 |
| 21-25 | **級数と漸近展開** | 収束判定法、冪級数（収束半径）、ランダウの記号、ロピタルの定理 |
| 26-29 | **総括と評価②** | 微分作用素、総括、教場試験（前期後半）と解説 |
| 30-33 | **多変数の微分（偏微分）** | 偏微分、全微分可能性、連鎖律、接平面と勾配ベクトル |
| 34-38 | **多変数の極値と凸性** | 多変数のテイラー展開、2次形式による極値判定、関数の凸性 |
| 39-42 | **陰関数と拘束付最適化** | 速度ベクトル、等位面、陰関数定理、ラグランジュの未定乗数法 |
| 43-44 | **中間評価③** | 後期前半の中間試験と解説 |
| 45-49 | **定積分の理論** | 面積の概念、微積分の基本定理、ダルブー和・リーマン和による定義 |
| 50-52 | **積分法の応用** | 不定積分の線形性、曲線の長さ、広義積分（ゼータ・ガンマ関数） |
| 53-56 | **重積分・微分方程式・評価** | 重積分、ヤコビアン（変数変換）、線形常微分方程式、最終試験 |
